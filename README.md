# Sorting Visualizer

A web-based sorting algorithm visualizer built with **HTML, CSS, and JavaScript**.  

## Features
- Visualize Bubble, Insertion, Selection, Merge, Quick Sort
- Smooth animations
- Dynamic array generation
- Color-coded sorting steps
